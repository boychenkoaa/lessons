## Доказать правильность

` {arr.length > 0} findMax(arr) {result = max(arr)} `

**Доказательство**

Инвариант: `I: 0 <= i <= n-1, result = max(arr[0:i])`

**База индукции**

n = 1, n = 2 -- очевидно

**Индукционный переход**

max(arr[0:i]) = max(max(0:i-1), arr[i]) -- очевидно

=> Функция корректна.
