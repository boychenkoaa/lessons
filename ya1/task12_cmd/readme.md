# Архитектура на командах

Слои.
1. Команды `pure_robot.py`
2. Командные обертки [robot_api.py](robot_api.py) `move_cmd`, `turn_cmd`, `set_state_cmd`, `start_cmd`, `stop_cmd`.
3. Аггрегирующие функции: `fold` -- более простая (неявно протаскивающая состояние) и `compose` -- генерирующая функцию-композицию.
4. В `main.py` результаты простого демо-прогона совпали
