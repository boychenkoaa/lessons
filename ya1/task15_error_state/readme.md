# Все пошло не по плану

Нужно добавить пайплайн обработки ошибок.

#### Модифицируем состояние робота
- вместо `log` ведем `infra = (ok, log)` (инфраструктура)
- добавляем количество воды и пены как `int`

#### Модифицируем монаду:
- `bind` игнорирует команды, если робот в ошибке
- `or_else` игнорирует команды, если робот не в ошибке

#### Модифицируем команды:
- добавляем декораторы на пред- и пост- условия, которые ставят робота в ошибку при их непрохождении `alarm_pre` и `alarm_post`
- добавляем запросы предусловий и постусловий (`robot_in_bounds`, `check_water`)
- оборачиваем пред- и пост- условия в декораторы внутри самих функций-команд
- добавляем команду `reset`, которая сбрасывает ошибку 
- добавляем команду `inc_water`, которая доливает воду

#### Модифицируем smoke-тест `main.py`
- Добавляем туда ошибку по воде, reset, долив воды и продолжение работы
