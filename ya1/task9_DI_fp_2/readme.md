# Задание 9: Dependency Injection в ФП -2

Организуем функциональную инъекцию одной функцией, а не пятью.

Робота, который строится по пяти функциям, делаем еще одной функцией.

В файлах:
- `make_robot` [robot_factory.py](robot_factory.py) -- фабрика функций-роботов, принимет 5 функций.
- `simple_robot` и `fast_robot` [robot_factory.py](robot_factory.py) -- функции-роботы. Результаты работы `make_robot`, обычный и быстрый робот.
- `RobotAPI` [robot_api.py](robot_api.py) -- класс с состоянием, принимающий функцию-робота и по сути обертка над ней. Хранит последнее состояние робота
